this 绑定丢失及修复

在 Javascript 中， this 关键字经常在我们不注意的时候就被错误引用，得到意料之外的结果，所以我自己对 this 引用丢失及修复的方式整理总结一下。

1 指向哪儿？

首先明确一下， this 是一个指向函数被调用时候的执行上下文的指针。

执行上下文中包含了函数的调用栈、函数调用方式、传入的参数等信息

this 的绑定发生在运行时，它指向什么完全取决于函数在哪里被调用
